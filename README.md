# Greenden-tailwind

Greenden Website
    Greenden is a modern, eco‑inspired website dedicated to promoting sustainable living and plant care.
    It showcases a curated collection of indoor and outdoor plants, gardening tips, and eco‑friendly practices — all designed to bring nature closer to people.

Project Overview
    Greenden is built using HTML, Tailwind CSS, and JavaScript.
    It features a clean, responsive design optimized for all devices, with sections including:

        Home – Welcoming visitors with a nature‑inspired hero banner.

        About Us – Sharing Greenden’s mission and values.

        Product Gallery – Displaying plants with names, prices, and care details.

        Footer – Featuring newsletter signup, contact info, and social links.


Tech Stack
    HTML5	Structure and layout
    Tailwind CSS	Styling and responsiveness
    GitHub	Version control and collaboration

Features
    Responsive design for all screen sizes.
    High‑resolution plant gallery.
    Smooth hover animations.
    Eco‑friendly theme and color palette.
    Fast loading and optimized images.
